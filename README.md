# Monolithic-Application
Monolithic Application with nodesJS 


NPM Packages to install 
dependencies": {
    "body-parser": "^1.20.0",
    "ejs": "^3.1.6",
    "express": "^4.17.3",
    "mongodb": "^4.5.0",
    "mongoose": "^6.2.10",
    "nodemon": "^2.0.15"
  }
  
  First install npm init 
  Secondly install all the packages above and run index.js
